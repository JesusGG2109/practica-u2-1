-- Objetivo del Sprint --

Desarrollar una página web interactiva utilizando HTML, CSS y JavaScript que incluya:

- Rótulo con nombre del equipo

- Sección con foto y descripción de los integrantes

- Botón para cambiar el color de fondo

- Alternancia entre integrantes

-- User Stories --
US1 – Identidad del equipo

Como usuario quiero ver el nombre del equipo y los integrantes
Para identificar quién desarrolló la página.

US2 – Cambio de color

Como usuario quiero hacer clic en un botón
Para que el fondo cambie de color dinámicamente.

US3 – Alternar integrantes

Como usuario quiero cambiar entre los integrantes
Para visualizar la información de ambos.
